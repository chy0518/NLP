# Order Matters Improvement

This repository contains our implementation and experiments for improving order robustness in multimodal large language models.

## Proposed Methods

- Permutation Self-Consistency
- Position-Calibrated Voting
- Query-Aware Relevance Reordering

## Structure

- `scripts/`: experiment scripts
- `data/`: data processing files or instructions
- `results/`: experimental results
- `docs/`: notes and paper-related documents
